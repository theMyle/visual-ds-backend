# Assessment Payload Specification

This document outlines the exact JSON structure expected by the `POST /api/assessments` endpoint.

## The Root Object: `Assessment`

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | `string` | **Yes** | Unique identifier for the assessment. (e.g., `"array-list-1"`) |
| `category` | `string` | **Yes** | The category the assessment belongs to. (e.g., `"array-list"`) |
| `questions` | `array` | **Yes** | A list of `Question` objects belonging to this assessment. |

---

## `Question` Object

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | `string` | No | Optional ID. If omitted, a secure UUID is automatically generated by the backend. |
| `text` | `string` | **Yes** | The actual question prompt. |
| `image_url` | `string` | No | Optional path/URL to an image. Can be omitted or set to `null`. |
| `type` | `string` | **Yes** | The type of question (e.g., `"multiple_choice"`, `"true_false"`). |
| `feedback` | `object` | **Yes** | A `Feedback` object containing responses for correct/incorrect answers. |
| `choices` | `array` | **Yes** | A list of `Choice` objects representing the possible answers. |

---

## `Feedback` Object

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `correct` | `string` | **Yes** | The message shown to the user if they pick the correct answer. |
| `incorrect` | `string` | **Yes** | The message shown to the user if they pick an incorrect answer. |

---

## `Choice` Object

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | `string` | No | Optional ID. If omitted, a secure UUID is automatically generated by the backend. |
| `text` | `string` | **Yes** | The text of the choice. |
| `is_correct`| `boolean`| **Yes** | `true` if this is the correct answer, `false` otherwise. |

---

### Full JSON Example (Cleaned)

```json
{
  "id": "array-list-1",
  "category": "array-list",
  "questions": [
    {
      "text": "How do you access the value (5) in this array nums?",
      "image_url": "/assessment/array-list/q1.png",
      "type": "multiple_choice",
      "feedback": {
        "correct": "That's right! Since array indices start at 0, the fifth element (the value 5) is at index 4.",
        "incorrect": "Not quite. Arrays start counting from 0, so the fifth element is actually at index 4."
      },
      "choices": [
        {
          "text": "nums[3]",
          "is_correct": false
        },
        {
          "text": "nums[4]",
          "is_correct": true
        }
      ]
    }
  ]
}
```
